About

The Orbit visualizes the video game scoring from various media. It has a standardized scale of these scores in graphs. The purpose of this project is to compare and contrast the evaluation of game media.


The Score

The Orbit shows the trend of video game taste. The core marks the scores from game media. The outer layer represent the public estimation.

All data collected from: IGN, GameSpot, Metascore, GamerSky, Steam(community)
